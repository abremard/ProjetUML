Instructions pour l'installation/compilation :

Methode 1 : IDE
1-Ouvir le projet dans un IDE
2-Compiler le projet

Methode 2 : En ligne de commande
1-Ouvrir une console de ligne de commande
2-Taper puis appuier sur la touche entrée :
$cd [lien vers le répertoire du projet]/ProjetUML
3-Taper puis appuier sur la touche entrée :
$make

Instructions pour l'utilisation :
1-Compiler le projet
2-Ouvrir une console de ligne de commande
3-Taper puis appuier sur la touche entrée :
$cd [lien vers le répertoire du projet]/ProjetUML
4-Taper puis appuier sur la touche entrée :
$./exec
5-Pour supprimer les éxecutables taper puis appuier sur la touche entrée :
$make clean
